# 房产投资计算器 - 需求文档

## 1. 概述

房产投资计算器是面向湾区高收入W2雇员的专业租房投资分析工具。现有"Brutal Calculator"电子表格已具备完整的税后收益模型，但存在于应用之外。将其迁移为原生交互工具，使其与家庭财务套件无缝集成，并支持家庭成员共享使用。

### 1.1 设计理念

- **纯前端计算**：所有计算逻辑在客户端完成，无需后端或数据库变更
- **实时响应**：任意输入变化立即触发全量重算，无需提交按钮
- **税务感知**：完整建模折旧税盾、被动损失递延、杠杆增值等高净值场景
- **基准对比**：以标普500长期年化收益（8%）为基准，评估房产投资性价比

### 1.2 访问路径

- **路由**：`/investments/property-calculator`
- **导航**：侧边栏新增「投资」标签（位于分析与设置之间），下设「房产计算器」导航链接

### 2.2 计算输出

#### 2.2.1 月度现金流

**计算公式**：
- 毛租金收入 = 月租金 × (1 − 空置率)
- 月供（本息）= PMT(年利率/12, 360, −贷款额)，贷款额 = 购买价格 × (1 − 首付比例)
- 月度房产税 = 购买价格 × 房产税率 / 12
- 月度保险 = 年度保险 / 12
- 物业费 = 月物业费（直接录入）
- 月度维护费 = 购买价格 × 1% / 12
- **实际月现金流** = 毛租金收入 − 月供 − 房产税 − 保险 − 物业费 − 维护费

**显示规则**：实际月现金流为负时显示红色，为正时显示绿色

#### 2.2.2 核心指标

| 指标 | 计算公式 |
|---|---|
| 投入资金 | 购买价格 × (首付比例 + 过户费率) |
| 年度现金流 | 实际月现金流 × 12 |
| 年度本金还款 | CUMPRINC(利率/12, 360, 贷款额, 1, 12, 0) 第一年 |
| **现金回报率 (CoC)** | 年度现金流 / 投入资金 |

**CoC颜色编码**：
- 红色：< 0%
- 黄色：0% ~ 4%
- 绿色：≥ 8%

#### 2.2.3 税后收益

| 指标 | 计算公式 |
|---|---|
| 年度折旧 | 购买价格 × (1 − 土地占比) / 27.5 |
| 第一年利息支出 | \|CUMIPMT(利率/12, 360, 贷款额, 1, 12, 0)\| |
| NOI | (毛租金收入 − 房产税 − 保险 − 物业费 − 维护费) × 12 |
| 纸面亏损 | NOI − 第一年利息 − 年度折旧 |
| **税务节省银行** | 纸面亏损 × 边际税率（仅纸面亏损 < 0 时显示，作为正向收益） |
| **有效收益率** | (年度现金流 − 税务节省 + 年度本金还款) / 投入资金 |

**注意事项**：税务节省银行属于「被动损失递延」，非W2收入直接抵扣项——除非用户具备房地产专业人士资格，否则该税务优惠将被暂时搁置，待出售时方可实现。页面须显示此说明。

#### 2.2.4 增值回报

| 指标 | 计算公式 |
|---|---|
| 年度增值收益 | 购买价格 × 预期增值率 |
| **增值回报率** | 年度增值收益 / 投入资金 |

#### 2.2.5 总财富回报率

- **总财富回报率 (Total Wealth ROI)** = 有效收益率 + 增值回报率

**颜色基准**：
- 红色：< 8%（低于标普500长期均值）—— 标签「低于大盘均值」
- 黄色：8% ~ 14%
- 绿色：≥ 15%（杠杆超越大盘）—— 标签「目标：超越大盘」

## 3. 业务规则

### 3.1 按揭计算

- 采用标准30年固定利率等额本息（360期）
- PMT 公式：`PMT(r, n, PV)` = `r × PV / (1 − (1 + r)^−n)`，其中 r = 年利率/12，n = 360

### 3.2 折旧规则

- 美国税法规定住宅房产折旧年限为 27.5 年
- 仅建筑物部分可折旧，土地不可折旧
- 折旧 = 购买价格 × (1 − 土地占比) / 27.5

### 3.3 税务节省展示规则

- 仅在「纸面亏损」< 0 时展示税务节省银行
- 展示为正向收益（绝对值 × 边际税率）
- 必须附带被动损失说明

### 3.4 颜色基准参考

| 指标 | 红色 | 黄色 | 绿色 |
|---|---|---|---|
| 月现金流 | < 0 | — | ≥ 0 |
| CoC 回报率 | < 0% | 0%–4% | ≥ 8% |
| 总财富回报率 | < 8% | 8%–14% | ≥ 15% |
